This repository is a practical guide for exploring and understanding data through coding. It is organized into six parts that cover the full workflow step by step:
Cleaning Part 1, Cleaning Part 2, Exploratory Data Analysis Part 1, Exploratory Data Analysis Part 2, Feature Engineering Part 1, and Feature Engineering Part 2.

The goal of this project is to build a clear personal reference for handling real datasets, starting from data cleaning and preprocessing,
moving through exploration and pattern discovery, and ending with feature engineering techniques that help prepare data for deeper analysis and modeling. 
It is designed to make revision easier and provide reusable examples for future projects.
